## Зависимости между модулями

| Модуль | Вызываемые модули/методы | Назначение вызова |
|--------|-------------------------|-------------------|
| Program.cs | InputModule.ReadString() | Получение строки от пользователя |
| Program.cs | ValidationModule.IsValidString() | Проверка, что строка не пустая |
| Program.cs | ProcessingModule.CountAndPrintVowels() | Подсчёт гласных и вывод результата |

## Примечания и улучшения
- Модульная структура позволяет легко менять вариант задания
- Можно добавить обработку разных языков
- Можно расширить ValidationModule для проверки длины строки

## Анализ модулей и зависимостей

### Входные данные
- InputModule.ReadString() → возвращает string

### Проверка данных
- ValidationModule.IsValidString() → принимает string, возвращает bool

### Обработка и вывод
- ProcessingModule.CountAndPrintVowels() → принимает string, выводит результат в консоль

### Схема передачи данных
Ввод (string) → Проверка (bool) → Обработка (void) → Вывод
